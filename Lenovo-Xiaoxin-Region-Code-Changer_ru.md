# 🔄 Смена региона Lenovo Xiaoxin (TB335FC и устройства на MTK)

Минималистичный, портативный и безопасный инструмент для изменения кода региона в разделе `proinfo` устройств на процессорах MediaTek (MTK). Изначально разработан для **Lenovo Xiaoxin Pad 2025 (TB335FC)**, но универсально совместим с другими устройствами MTK (UFS/EMMC) при наличии правильного scatter-файла.

> ⚠️ **ОТКАЗ ОТ ОТВЕТСТВЕННОСТИ: ВСЕ ДЕЙСТВИЯ ВЫ ВЫПОЛНЯЕТЕ НА СВОЙ СТРАХ И РИСК!**  
> Изменение раздела `proinfo` несет риск превращения устройства в "кирпич" при некорректном использовании.  
> **Важно:** Смена региона с высокой вероятностью **отключит получение OTA-обновлений**. Всегда делайте полный бэкап перед началом работы.

---

## ✨ Возможности

- 🚀 **Без установки:** Полностью портативный. Просто распакуйте и запустите.
- 🛡️ **Безопасность и валидация:** Проверка размера файла, строгая проверка ввода (ровно 2 латинские буквы) и контроль измененных байт перед сохранением.
- 🌐 **Двуязычность:** Полная поддержка русского и английского интерфейсов.
- 🧠 **Умное определение:** Автоматически извлекает правильный адрес `proinfo` и тип памяти (UFS/EMMC) из вашего scatter-файла.
- ⚡ **Без внешних зависимостей:** Использует встроенный PowerShell Windows для логики патчинга. Не требует Python или сторонних библиотек.
- 🔗 **Интеграция с SP Flash Tool:** Пошаговые инструкции и возможность запуска прошивальщика прямо из меню.

---

## 📋 Требования

1. **Windows 10 / 11** (со встроенным PowerShell 5.1+).
2. Установленные **драйверы MediaTek VCOM**.
3. **SP Flash Tool** (рекомендуется версия 6.x).
4. **Scatter-файл** (`MTxxxx_Android_scatter.xml`) из прошивки вашего устройства.

## 🚀 Как использовать

1. Поместите **scatter-файл** вашего устройства в папку `image`.
2. Корневая папка уже содержит **`SP Flash Tool`**.
3. Запустите `XiaoxinRegionCodeChanger_RU.bat` (или `_EN.bat`).
4. Следуйте рекомендуемому рабочему процессу:
   - **Шаг 1:** `1` → Найти адрес `proinfo` в scatter-файле.
   - **Шаг 2:** `2` → Сделать дамп (чтение) раздела `proinfo` через SP Flash Tool.
   - **Шаг 3:** `4` → Запатчить полученный файл, введя желаемый 2-буквенный код региона (например, `GB`, `RU`, `US`).
   - **Шаг 4:** `5` → Записать пропатченный файл обратно в устройство через SP Flash Tool.
   - **Шаг 5:** `6` → В любой момент просмотреть информацию об устройстве и параметрах раздела из scatter-файла.

---

## ⚠️ Важные примечания

- Инструмент принимает **любые 2 латинские буквы** (например, `GB`, `XY`). Он найдет *любой* существующий токен формата `XXYY` в файле и заменит его. Если валидный токен не будет найден, процесс безопасно прервется.
- После прошивки нового региона перейдите в **Настройки → Приложения → Google Play Маркет → Хранилище → Очистить данные**, чтобы изменения вступили в силу.
- **Никогда** не отключайте USB-кабель во время процессов чтения или записи.

## 🛠️ Решение проблем

| Проблема | Решение |
|----------|---------|
| `❌ Код должен состоять из 2 латинских букв.` | Вы ввели цифры, кириллицу или оставили поле пустым. Введите ровно 2 английские буквы (например, `RU`). |
| `❌ Scatter-файл не найден` | Убедитесь, что ваш scatter-файл (`.txt` или `.xml`) находится внутри папки `image`. |
| `❌ Токен региона не найден в файле` | Файл `proinfo` может быть поврежден, пуст или снят с другого устройства. Сделайте дамп заново через SP Flash Tool. |
| `SPFlashToolV6.exe не найден` | Скачайте `SP Flash Tool` и распакуйте его в ту же папку, где лежат `.bat` файлы. |
