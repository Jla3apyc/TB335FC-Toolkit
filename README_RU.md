# 🛠️ Lenovo Xiaoxin Pad (TB335FC) Toolkit & Guides

Коллекция утилит и подробных руководств для управления, модификации и прошивки устройств на базе MediaTek (MT6835), с фокусом на **Lenovo Xiaoxin Pad 2025 (TB335FC)**.

> ⚠️ **ВАЖНО:** Все действия вы выполняете на свой страх и риск. Модификация системных разделов может привести к потере гарантии или превращению устройства в «кирпич». **Всегда делайте резервные копии перед началом работы.**

---

## 📚 Проекты

Этот набор инструментов состоит из взаимосвязанных проектов. Выберите тот, который соответствует вашей задаче:

### 📖 1. [TB335FC flash guide](TB335FC-flash-guide_ru.md)
Пошаговое руководство по полной прошивке, восстановлению и обслуживанию устройства с помощью SP Flash Tool.
* **Для чего:** Установка глобальной прошивки TB336FU на китайский планшет TB335FC, восстановление устройства после неудачных модификаций (unbrick).
* **Особенности:** Используется новый авторский способ установки глобальной прошивки китайский планшет.

### 🌍 2. [TB335FC region code changer](TB335FC-region-code-changer_ru.md)
Инструмент для изменения кода региона в разделе `proinfo`.
* **Для чего:** Активация Widevine L1 (HD-видео), сертификация Google Play, смена региональных настроек.
* **Особенности:** Не требует Python, работает через встроенный PowerShell. Автоматически читает scatter-файлы.

### 🧹 3. [TB336FU bloatware uninstaller](TB336FU-bloatware-uninstaller_ru.md)
Скрипт для отключения OTA-обновлений и удаления предустановленного мусорного ПО.
* **Для чего:** В автоматическом режиме отключает неработающие OTA-обновления и облегчает удаление предустановленного мусорного ПО (bloatware) с Lenovo Xiaoxin Pad 2025 (прошивка TB336FU) через ADB.
* **Особенности:** Разработан для прошивки TB336FU ZUI_17.5.10.213, установленной на планшет TB335FC. Использует безопасное удаление через `pm uninstall --user 0` с возможностью полного восстановления приложений. Легко настраивается редактированием списка пакетов в `.bat`-файле.
